# Why

Managing a complex ESLint configuration takes time and effort.
**Sheriff does it for you.**

There are plenty of factors to consider when you want to tackle a complex ESLint configuration.<br />
It requires a lot of expertise into JavaScript engines
and the multitude of JavaScript libraries, their nuances, and their interlacing.

_And_, it’s a huge burden to maintain.

With Sheriff, maintaining the ESLint configuration is as simple as updating Sheriff to the latest version.

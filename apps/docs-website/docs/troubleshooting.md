# 🩹 Troubleshooting

## The command `yarn create @sherifforg/config` fails when using yarn

Depending on the context and under specific conditions,
the setup command may fail to install the dependencies when using `yarn`.<br />
If that happens to you, you can install them yourself.
The `yarn create @sherifforg/config` command should tell you the correct command
for you to run so at the end of the process.
If, for some reason, that does not happen, refer to the [manual setup instructions](./setup/manual-setup.mdx).

Alternatively, consider switching your package manager to [pnpm](https://pnpm.io/).

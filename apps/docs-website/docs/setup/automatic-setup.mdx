# Automatic setup (recommended)

Let the CLI take care of everything!

1. Run this command in your terminal:

  ```bash npm2yarn
  npm init @sherifforg/config
  ```

  :::tip

  For monorepos,
  refer to the extra details in the [monorepo support](../monorepo-support.mdx#setup-with-npm-init-sherifforgconfig) guide.

  :::

2. [Setup VS Code support](../vscode-support.md) (_optional_)

…and you are good to go! Happy hacking 🎉!
